# LemonKit4Android
一个让你开发在构建Android程序时候更加迅速、UI更加美观的框架。假如你是一个iOS开发者，你可能只需要学习Java的语法，就可以基于LK4A进行Android开发了。

iOS OC版本：https://github.com/1em0nsOft/LemonKit4iOS

iOS Swift版本：https://github.com/1em0nsOft/LemonKit4iOS-Swift
